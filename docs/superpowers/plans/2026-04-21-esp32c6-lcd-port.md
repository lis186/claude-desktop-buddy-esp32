# ESP32-C6-Touch-LCD-1.47 Port Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a new sibling PlatformIO project at `~/Documents/claude-desktop-buddy-lcd` that runs the existing claude-desktop-buddy on the Waveshare ESP32-C6-Touch-LCD-1.47 board. BLE wire protocol is unchanged; desktop pairing works identically.

**Architecture:** Start by copying the current project tree, delete the features the C6 cannot support (PSRAM-dependent GIF characters, audio codec, AXP PMU, TCA9554 expander), rewrite the display / touch / power / input HAL for the C6's specific peripherals (ST7789 172×320 LCD, AXS5106L touch, QMI8658 IMU, GPIO0 ADC battery, single BOOT key). The board-agnostic 184×224 logical canvas is preserved — `hwDisplayPush` crops to the 172-wide screen and blits at y=96, leaving the top 96 px for a dedicated status bar with clock/battery/BLE/HUD.

**Tech Stack:** PlatformIO (pioarduino ESP32 platform 53.x) · Arduino framework 3.x · ESP32-C6 · Arduino_GFX (ST7789 HWSPI) · AXS5106L driver (vendored) · SensorLib (QMI8658) · NimBLE-Arduino 2.x (bumped from 1.4 for C6 support) · ArduinoJson 7 · U8g2

**Source references:**
- Spec: `docs/superpowers/specs/2026-04-21-esp32c6-lcd-port-design.md`
- Vendor demo: `~/Downloads/ESP32-C6-Touch-LCD-1.47-Demo/`
- Schematic: `~/Downloads/ESP32-C6-Touch-LCD-1.47-Demo/ESP32-C6-Touch-LCD-1.47-Schematic.pdf`
- Source project: `~/Documents/claude-desktop-buddy/` (kept untouched by this plan)

**Note on verification:** This is a hardware port; there are no unit tests. Each task ends with `pio run` + a physical flash + a specific observable check, then a commit. Tasks that are code-only (no observable change yet) end with a build-only verify.

bool hwDisplayInit() {
  s_bus = new Arduino_HWSPI(PIN_LCD_DC, PIN_LCD_CS, PIN_LCD_SCLK, PIN_LCD_MOSI);
  s_gfx = new Arduino_ST7789(
    s_bus, PIN_LCD_RST, 0, false,
    BOARD_SCREEN_W, BOARD_SCREEN_H,
    BOARD_ST7789_COL_OFFSET, 0,
    BOARD_ST7789_COL_OFFSET, 0);
  if (!s_gfx->begin()) { Serial.println("hwDisplay: gfx begin failed"); return false; }
  lcd_reg_init(s_bus);
  s_gfx->setRotation(0);
  s_gfx->fillScreen(RGB565_BLACK);

  ledcAttach(PIN_LCD_BL, 5000, 8);
  ledcWrite(PIN_LCD_BL, BRIGHT_LUT[s_brightLvl]);

// src/hw/input.cpp (full replacement of the prior placeholder)
#include "input.h"
#include "pins.h"
#include "display.h"
#include <Arduino.h>
#include <Wire.h>
#include <esp_lcd_touch_axs5106l.h>

static const uint32_t KEY_DEBOUNCE_MS     = 25;
static const uint32_t KEY_LONG_PRESS_MS   = 1000;
static const uint32_t KEY_RESTART_MS      = 5000;

static const int16_t SWIPE_PIX_THRESHOLD  = 30;   // pixels along dominant axis
static const uint32_t TAP_MAX_MS          = 250;

static uint32_t s_keyPressedAt  = 0;
static bool     s_keyDown       = false;
static bool     s_keyLongFired  = false;

static bool     s_touchDown     = false;
static int16_t  s_touchStartX   = 0;
static int16_t  s_touchStartY   = 0;
static int16_t  s_touchLastX    = 0;
static int16_t  s_touchLastY   = 0;
static uint32_t s_touchStartMs  = 0;

void hwInputInit() {
  pinMode(PIN_KEY1, INPUT_PULLUP);
  bsp_touch_init(&Wire, PIN_TP_RST, PIN_TP_INT, 0, BOARD_SCREEN_W, BOARD_SCREEN_H);
}

bool hwInputTouchActive() { return s_touchDown; }

// Map screen-space (sx 0..171, sy 0..319) to canvas-space (cx 0..183, cy 0..223).
// Taps where sy < BOARD_CANVAS_OFFSET_Y are in the status-bar region and are
// currently not forwarded to main (reserved for future statusbar shortcuts).
static bool mapToCanvas(int16_t sx, int16_t sy, int16_t* cx, int16_t* cy) {
  if (sy < BOARD_CANVAS_OFFSET_Y) return false;
  *cx = sx + BOARD_CANVAS_CROP_X;
  *cy = sy - BOARD_CANVAS_OFFSET_Y;
  return true;
}

static InputEvent classifySwipe(int16_t dx, int16_t dy) {
  if (abs(dx) < SWIPE_PIX_THRESHOLD && abs(dy) < SWIPE_PIX_THRESHOLD) return EV_NONE;
  if (abs(dx) > abs(dy)) {
    return dx > 0 ? EV_GESTURE_SWIPE_RIGHT : EV_GESTURE_SWIPE_LEFT;
  }
  return dy > 0 ? EV_GESTURE_SWIPE_DOWN : EV_GESTURE_SWIPE_UP;
}

InputEvent hwInputPoll(TapInfo* out) {
  uint32_t now = millis();

  // --- button (GPIO9, active-low) ---
  int keyLvl = digitalRead(PIN_KEY1);
  if (keyLvl == LOW) {                         // pressed
    if (!s_keyDown) {
      s_keyDown = true;
      s_keyLongFired = false;
      s_keyPressedAt = now;
    } else {
      uint32_t held = now - s_keyPressedAt;
      if (held >= KEY_RESTART_MS) {
        Serial.println("input: 5s hold on BOOT — restarting");
        delay(50);
        ESP.restart();
      }
    }
  } else {                                     // released
    if (s_keyDown) {
      uint32_t held = now - s_keyPressedAt;
      s_keyDown = false;
      if (held < KEY_DEBOUNCE_MS) { /* glitch */ }
      else if (held < KEY_LONG_PRESS_MS) return EV_BTN_A_PRESS;
      else return EV_BTN_A_LONG_PRESS;
    }
  }

  // --- touch ---
  bsp_touch_read();
  touch_data_t td;
  bool pressed = bsp_touch_get_coordinates(&td);

  if (pressed) {
    int16_t sx = td.coords[0].x;
    int16_t sy = td.coords[0].y;
    if (!s_touchDown) {
      s_touchDown    = true;
      s_touchStartX  = sx;
      s_touchStartY  = sy;
      s_touchStartMs = now;
    }
    s_touchLastX = sx;
    s_touchLastY = sy;
  } else if (s_touchDown) {
    // finger lifted — classify gesture
    s_touchDown = false;
    int16_t dx = s_touchLastX - s_touchStartX;
    int16_t dy = s_touchLastY - s_touchStartY;
    uint32_t dur = now - s_touchStartMs;

    InputEvent gest = classifySwipe(dx, dy);
    if (gest != EV_NONE) return gest;

    if (dur <= TAP_MAX_MS) {
      // tap — map to canvas space and emit EV_TAP with both current and start coords
      int16_t cx, cy, scx, scy;
      if (mapToCanvas(s_touchLastX, s_touchLastY, &cx, &cy) &&
          mapToCanvas(s_touchStartX, s_touchStartY, &scx, &scy)) {
        if (out) {
          out->x      = cx;
          out->y      = cy;
          out->startX = scx;
          out->startY = scy;
        }
        return EV_TAP;
      }
    }
  }

  return EV_NONE;
}

// src/hw/statusbar.cpp
#include "statusbar.h"
#include "pins.h"
#include "display.h"
#include <Arduino.h>
#include <Arduino_GFX_Library.h>

// 172*96 = 16512 px * 2 B = 33 KB DRAM.
static uint16_t s_fb[BOARD_SCREEN_W * BOARD_STATUSBAR_H];
static bool     s_dirty = true;

static int     s_hour = 0, s_minute = 0;
static uint8_t s_bat = 100;
static bool    s_charging = false;
static uint8_t s_ble = 0;
static HudMode s_hud = HUD_BLANK;
static uint8_t s_sessionCount = 0;
static uint16_t s_tokensPct   = 0;

static uint32_t s_lastClockMs     = 0;
static uint32_t s_lastBleBatMs    = 0;
static uint32_t s_lastHudMs       = 0;
static uint8_t  s_pulsePhase      = 0;   // 0/1, toggles at 2 Hz for attention pulse

// Simple 5x7 digit glyphs (renders HH:MM in the top-left)
// Each byte is one column (5 columns per glyph), LSB = top row.
static const uint8_t DIGIT5x7[11][5] = {
  {0x3E,0x51,0x49,0x45,0x3E}, // 0
  {0x00,0x42,0x7F,0x40,0x00}, // 1
  {0x42,0x61,0x51,0x49,0x46}, // 2
  {0x21,0x41,0x45,0x4B,0x31}, // 3
  {0x18,0x14,0x12,0x7F,0x10}, // 4
  {0x27,0x45,0x45,0x45,0x39}, // 5
  {0x3C,0x4A,0x49,0x49,0x30}, // 6
  {0x01,0x71,0x09,0x05,0x03}, // 7
  {0x36,0x49,0x49,0x49,0x36}, // 8
  {0x06,0x49,0x49,0x29,0x1E}, // 9
  {0x00,0x36,0x36,0x00,0x00}, // ':' (index 10)
};

static inline uint16_t* px(int x, int y) { return &s_fb[y * BOARD_SCREEN_W + x]; }

static void drawGlyph(int x, int y, uint8_t g, uint16_t fg, uint16_t bg, int scale) {
  for (int col = 0; col < 5; ++col) {
    uint8_t bits = DIGIT5x7[g][col];
    for (int row = 0; row < 7; ++row) {
      uint16_t c = (bits & (1 << row)) ? fg : bg;
      for (int sy = 0; sy < scale; ++sy)
        for (int sx = 0; sx < scale; ++sx) {
          int px_x = x + col * scale + sx;
          int px_y = y + row * scale + sy;
          if (px_x >= 0 && px_x < BOARD_SCREEN_W && px_y >= 0 && px_y < BOARD_STATUSBAR_H)
            *px(px_x, px_y) = c;
        }
    }
  }
}

static void drawClock(int x0, int y0, int hour, int minute, uint16_t fg, uint16_t bg) {
  int d[5] = { hour / 10, hour % 10, 10, minute / 10, minute % 10 };
  for (int i = 0; i < 5; ++i) drawGlyph(x0 + i * (5 * 2 + 1), y0, d[i], fg, bg, 2);
}

static void drawNN(int x0, int y0, uint8_t pct, uint16_t fg, uint16_t bg) {
  uint8_t d[2] = { (uint8_t)((pct / 10) % 10), (uint8_t)(pct % 10) };
  drawGlyph(x0,             y0, d[0], fg, bg, 1);
  drawGlyph(x0 + 6,         y0, d[1], fg, bg, 1);
}

static void drawBleDot(int cx, int cy) {
  uint16_t c;
  switch (s_ble) {
    case 3: c = 0x07E0; break;                             // connected = green
    case 2: c = (s_pulsePhase ? 0x001F : 0x0000); break;   // pairing = blue pulse
    case 1: c = 0xC618; break;                             // advertising = light grey
    default: c = 0xF800 >> 3;                              // no client = dim red
  }
  for (int dy = -3; dy <= 3; ++dy)
    for (int dx = -3; dx <= 3; ++dx)
      if (dx*dx + dy*dy <= 9) *px(cx + dx, cy + dy) = c;
}

static void clearFb(uint16_t bg) {
  for (int i = 0; i < BOARD_SCREEN_W * BOARD_STATUSBAR_H; ++i) s_fb[i] = bg;
}

static void redrawTopStrip() {
  // Row 0..23 = 24 px
  // Clear the top strip only
  for (int y = 0; y < 24; ++y)
    for (int x = 0; x < BOARD_SCREEN_W; ++x) *px(x, y) = 0x0000;
  drawClock(4, 4, s_hour, s_minute, 0xFFFF, 0x0000);
  drawBleDot(BOARD_SCREEN_W / 2, 12);
  drawNN(BOARD_SCREEN_W - 16, 8, s_bat > 99 ? 99 : s_bat, s_charging ? 0x07E0 : 0xFFFF, 0x0000);
}

static void redrawHudStrip() {
  // Row 24..95 = 72 px
  uint16_t bg = 0x0000;
  if (s_hud == HUD_ATTENTION && s_pulsePhase) bg = 0xF800;  // full red pulse
  for (int y = 24; y < BOARD_STATUSBAR_H; ++y)
    for (int x = 0; x < BOARD_SCREEN_W; ++x) *px(x, y) = bg;
  if (s_hud == HUD_BUSY) {
    // horizontal progress bar at y=60..66
    int w = (s_tokensPct > 1000) ? BOARD_SCREEN_W : (BOARD_SCREEN_W * s_tokensPct) / 1000;
    for (int y = 60; y < 66; ++y)
      for (int x = 0; x < w; ++x) *px(x, y) = 0x07FF;   // cyan
    // session count digits
    drawNN(4, 36, s_sessionCount, 0xFFFF, 0x0000);
  }
}

void statusbarInit() {
  clearFb(0x0000);
  s_dirty = true;
}

void statusbarSetClock(int h, int m) {
  if (h == s_hour && m == s_minute) return;
  s_hour = h; s_minute = m;
  s_dirty = true;
}

void statusbarSetBattery(uint8_t p, bool chg) {
  if (p == s_bat && chg == s_charging) return;
  s_bat = p; s_charging = chg;
  s_dirty = true;
}

void statusbarSetBle(uint8_t st) {
  if (st == s_ble) return;
  s_ble = st; s_dirty = true;
}

void statusbarSetHud(HudMode m, uint8_t sc, uint16_t tp) {
  if (m == s_hud && sc == s_sessionCount && tp == s_tokensPct) return;
  s_hud = m; s_sessionCount = sc; s_tokensPct = tp;
  s_dirty = true;
}

void statusbarTick() {
  uint32_t now = millis();
  if (now - s_lastClockMs >= 1000) { s_lastClockMs = now; s_dirty = true; }
  if (now - s_lastBleBatMs >= 5000) { s_lastBleBatMs = now; s_dirty = true; }
  if ((s_hud == HUD_ATTENTION || s_ble == 2) && (now - s_lastHudMs >= 250)) {
    s_lastHudMs = now;
    s_pulsePhase ^= 1;
    s_dirty = true;
  }
  if (s_dirty) {
    redrawTopStrip();
    redrawHudStrip();
  }
}

bool statusbarDirty() { return s_dirty; }

void statusbarBlit() {

### Task 12: Power: ADC Battery + Auto-Off Timer

**Files:**

- [ ] **Step 1: Replace power.cpp**

```cpp
// src/hw/power.cpp
#include "power.h"
#include "pins.h"
#include <Arduino.h>

static float    s_lastVbat = 4.0f;
static uint32_t s_lastSampleMs = 0;

static const float V_EMPTY = 3.0f;
static const float V_FULL  = 4.2f;
static const float V_USB   = 4.05f;   // hysteresis-free threshold for "charging" heuristic

void hwPowerInit() {
  pinMode(PIN_BAT_ADC, INPUT);
  analogReadResolution(12);
  analogSetPinAttenuation(PIN_BAT_ADC, ADC_11db);
}

static void sampleIfStale() {
  uint32_t now = millis();
  if (now - s_lastSampleMs < 200) return;
  s_lastSampleMs = now;
  uint32_t mv = analogReadMilliVolts(PIN_BAT_ADC);
  float v = (float)mv * BOARD_BAT_ADC_MULT / 1000.0f;
  // 1-pole IIR smoothing so the HUD doesn't jitter a point every tick
  s_lastVbat = (s_lastVbat * 0.8f) + (v * 0.2f);
}

float hwPowerBatteryVolts() {
  sampleIfStale();
  return s_lastVbat;
}

uint8_t hwPowerBatteryPercent() {
  float v = hwPowerBatteryVolts();
  if (v <= V_EMPTY) return 0;
  if (v >= V_FULL)  return 100;
  float pct = (v - V_EMPTY) * 100.0f / (V_FULL - V_EMPTY);
  return (uint8_t)(pct + 0.5f);
}

bool hwPowerIsCharging() {
  return hwPowerBatteryVolts() > V_USB;
}

bool hwPowerIsUSB() {
  return hwPowerIsCharging();
}

void hwPowerShutdown() {
  // No PMU — the board cannot cut its own power. Restart is the strongest
  // action available; the caller's intent (user held BOOT for 5 s) is
  // "turn it off, I'll deal with it", so at minimum get back to a clean boot.
  ESP.restart();
}
```

The existing `ble_bridge.cpp` uses NimBLE-Arduino 1.4 API. Key differences in 2.x:

| 1.x | 2.x |
|---|---|
| `BLEDevice::init()` returns void | same |
| `BLECharacteristicCallbacks::onWrite(BLECharacteristic* c)` | `onWrite(BLECharacteristic* c, ble_gap_conn_desc* desc)` |

| `BLE2902` descriptor no longer manually added | NimBLE handles CCCD automatically |
| Header `<BLE2902.h>` no longer exists | remove include |

## Status Bar

The top 96 px of the screen is a dedicated HUD:

- **Top strip (0–23):** clock HH:MM · BLE indicator dot · battery %

## Sleep

Only the LCD panel sleeps. MCU and BLE stay online always, so the device is instantly responsive when you wake it. Battery life on a 500 mAh cell is ~10–15 h idle.

| State | Auto-off |
|---|---|
| USB plugged | never |

| Approval prompt up | never |

## Limitations vs AMOLED boards

- **No custom GIF characters** — ASCII species only (all 18 species, all 7 animations)
- **No audio** — no hover tones, no confirmation blips
- **No hardware power-off** — 5 s BOOT hold restarts the device (closest equivalent)
- **Single button** — touch is primary for "B"-equivalent actions (deny, transcript scroll)
- **Slightly slower animations** — single-core 160 MHz hits ~15–20 FPS vs S3's 25–30
